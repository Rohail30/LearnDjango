Setting up Django!

Setting up Django on your computer is a breeze! Just follow these basic steps. However, make sure you have python and pip installed.

To check, type "python --version" and "pip --version." n command prompt. Easy, right?





First, create a folder with any name you like and place it wherever you want. I'm choosing to name my folder "Django." Open your terminal and move to the directory where you've set up your folder. I have created on desktop. Alright, let's dive in. In the terminal, please ignore the quotation marks and type the following commands:

  1. "python -m virtualenv env."

  2. "cd env/Scripts"

  3. "activate"

  4. "pip install django"

  5. "python", "import django", "django.__version__", "ctr z"

  6. Now go back to the folder. "cd ..", "cd .."

  7. "django-admin startproject MyDjangoProject", "cd MyDjangoProject", "python manage.py startapp home"

  8. This is just open your project folder in VS Code. you can skip this step and open your project manualy. "code ."


  9. Finally, to start server. "python manage.py runserver"
